# η ποσότητα
## количество
- Ед: ποσότητα, ποσότητα, ποσότητας
- Мн: ποσότητες, ποσότητες, ποσοτήτων

### Примеры
*В небольших количествах это полезно.* = *Σε μικρές ποσότητες αυτό είναι χρήσιμο.*
*Нам нужно большое количество воды.* = *Μας χρειάζεται μεγάλη ποσότητα νερού.*
*Я покупаю небольшое количество кофе.* = *Αγοράζω μικρή ποσότητα καφέ.*

# η υπομονή
## терпение
- Ед: υπομονή, υπομονή, υπομονής
- Мн: υπομονές, υπομονές, υπομονών

### Примеры
*Нужно иметь терпение.* = *Πρέπει να έχουν υπομονή.*
*Для такой работы нужно большое терпение.* = *Για τέτοια δουλειά χρειάζεται μεγάλη υπομονή.*
*Учитель объясняет всё с терпением.* = *Ο δάσκαλος εξηγεί τα πάντα με υπομονή.*
