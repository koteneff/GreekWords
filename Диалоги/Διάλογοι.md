# η ιδέα
## идея
- Ед: η ιδέα / της ιδέας / την ιδέα
- Мн: οι ιδέες / των ιδεών / τις ιδέες

### Примеры
*Δεν νομίζω ότι είναι καλή ιδέα.* = *Не думаю, что это хорошая идея.*
*Έχει μια ωραία ιδέα για τη συζήτηση.* = *У него есть хорошая идея для обсуждения.*
*Σκεφτήκαμε πολλές ιδέες πριν αποφασίσουμε.* = *Мы обдумали много идей, прежде чем решить.*

# νομίζω
## думаю; считаю

*νομίζω, νόμισα, θα νομίσω*
- νομίζω / νομίζεις / νομίζει / νομίζουμε / νομίζετε / νομίζουν
- νόμισα / νόμισες / νόμισε / νομίσαμε / νομίσατε / νόμισαν
- θα νομίσω / θα νομίσεις / θα νομίσει / θα νομίσουμε / θα νομίσετε / θα νομίσουν

### Примеры
*Δεν νομίζω ότι είναι καλή ιδέα.* = *Не думаю, что это хорошая идея.*
*Νομίζω ότι θα τα βρούμε.* = *Я думаю, что мы договоримся.*
*Νομίζεις ότι αυτό είναι σωστό;* = *Ты думаешь, что это правильно?*

# σίγουρος
## уверенный
- Ед: σίγουρος / σίγουρη / σίγουρο
- Мн: σίγουροι / σίγουρες / σίγουρα

### Наречие
- σίγουρα - уверенно; точно
- πιο σίγουρα - увереннее

### Родственные слова:
- η σιγουριά = уверенность

### Примеры
*Δεν είμαι σίγουρος.* = *Я не уверен.*
*Είσαι σίγουρη ότι είναι καλή ιδέα;* = *Ты уверена, что это хорошая идея?*
*Τώρα είμαστε πιο σίγουροι για την απόφασή μας.* = *Теперь мы более уверены в своём решении.*
